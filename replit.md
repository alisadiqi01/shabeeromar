# Shabir Omar Plastics — شرکت تولیدی شبیر عمر

A luxury RTL landing page for **Shabir Omar** (شبیر عمر پلاستیک), an Afghan PVC/PU footwear manufacturer based in Kabul. Built with TanStack Start + React 19 + Tailwind CSS v4, originally generated by Lovable.

## Stack

- **Framework**: TanStack Start (SSR-capable React meta-framework)
- **Styling**: Tailwind CSS v4 with a custom dark gold/luxury design system
- **Language**: TypeScript
- **Package manager**: Bun
- **UI components**: shadcn/ui (Radix primitives)
- **Routing**: TanStack Router (file-based, `src/routes/`)

## Running the app

```bash
bun run dev   # starts on http://localhost:5000
```

The workflow "Start application" is configured to run `bun run dev` on port 5000.

## Project structure

```
src/
  routes/         # File-based routes (TanStack Router)
    __root.tsx    # App shell (HTML, fonts, meta tags)
    index.tsx     # Main landing page (all sections)
  components/ui/  # shadcn/ui component library
  styles.css      # Tailwind + custom design tokens (gold palette, glass utilities)
  server.ts       # SSR middleware wrapper
public/
  images/         # Static product images (queen-010.png, etc.)
  favicon.ico
```

## Design system

The site uses a custom dark luxury palette (deep navy + gold) defined in `src/styles.css`. Key utilities: `.glass`, `.glass-gold`, `.btn-gold`, `.text-gold`, `.animate-float`, `.animate-rise`.

## Content

All content is in Persian/Dari (RTL). The single-page app covers:
- Hero with product image
- Product categories and cards
- About / factory sections
- WhatsApp-based order form (sends via `wa.me`)
- Contact details (Kabul address + phone numbers)
